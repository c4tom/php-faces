@pattern Tag creates a template for ui components and <@ui> is used with Example

name : name of your  pattern

prefix : name space used by import

extends : a component

@ui Implements a pattern

Name : a pattern name 

Sample 1
     <@import prefix="f" taglib="phpf.ui.*"/>
<@pattern name="mytag"
            prefix="f"
            extends="button"
            text="My text"
            style="color:navy;font-weight:bold;" 
            onclick="ajaxevent"/>




it is used @ui




 <@ui name="mytag1" pattern="mytag"/>
 <@ui name="mytag2" pattern="mytag"/>
 <@ui name="mytag3" pattern="mytag" text="orther text"/>
