@htmlpattern Tag creates a template for html and <@html> used

Name : name of your  pattern

@html Implements a htmlpattern

Name : a pattern name 

Sample 1
<@htmlpattern name="html">
           <div style="color:red;font-weight:bold;" > Own Text  </div>
  </@htmlpattern>
it is used
 
<@html pattern="html"/>
 
Out:
<div style="color:red;font-weight:bold;" > Own Text  </div>
 
Sample 2

        <div style="color:red;font-weight:bold;" > test html pattern{$i} </div>

<c:for var="$i" begin="1" to="10" step="1">
